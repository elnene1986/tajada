# Screenshot Captions — Tajada

App Store accepts up to 10 screenshots per device size. These captions are designed for the **6.7" iPhone** screenshot set (the size Apple requires; smaller sizes can reuse). Each caption is the short headline that overlays the screenshot — keep them under ~30 characters so they read clearly at thumbnail size.

The order matters: most users only see screenshots 1-3 in the listing's default scroll. Front-load the value prop.

## Production notes for the screenshots themselves

- **Frame:** put screenshots inside the standard iPhone 15 Pro frame, or use a clean device-less rectangle. Apple has no opinion on this; consistency matters more than the choice.
- **Background:** the Tajada paper-tone (`#F2E9D8`) works as a brand-consistent backdrop. Avoid bright white — it'll wash out the saffron accent.
- **Caption typography:** SF Pro or any clean sans-serif. Bold for the main caption, regular weight for the subcaption.
- **Caption color:** Tajada ink (`#14100C`) on the paper background.
- **Status bar:** show the iOS status bar with a real-looking time ("9:41" is Apple's traditional dummy) and full battery. No carrier logo.
- **Demo data:** use fake but realistic data — "Patreon Inc · $940", "Stripe Payments · $1,284", etc. Don't use real personal data, and don't use placeholder values like "$0.00" or "Lorem ipsum".
- **For the Spanish listing,** every label visible in the screenshot should be Spanish. Apple's review team has rejected listings where the screenshot UI didn't match the listing language.
